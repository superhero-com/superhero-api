# Bonding Curve TokenSale launchpad API

## Requirements
```
node >= 18
postgres >=16
redis
```

## Installation

```bash
$ npm install
```

## Update ENV configs
```bash
$ cp .env.example .env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
